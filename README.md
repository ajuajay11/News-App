# React News App

A simple news application built with **React + TypeScript**, **Redux Toolkit (RTK Query)**, and **i18n** supporting **English & Arabic**.  
The app fetches news data from JSONPlaceholder and demonstrates clean state management, responsive UI, and loading skeletons.

---

## Tech Stack
- React
- TypeScript
- Redux Toolkit + RTK Query
- React Router
- Material UI
- i18next
- Tailwind CSS

---

## Features
- News listing & post details
- Dynamic images with fallback
- Skeleton loaders
- Light / Dark theme
- Language switch (EN / AR)
- Responsive layout
